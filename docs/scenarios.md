# Scenarios

## Scenario 1: CSV-based Analysis

### Input
Local CSV file containing tourism data

### Process
- Read file
- Compute KPIs

### Output
Structured report with:
- Average occupancy rate
- Total revenue
- Top regions

---

## Scenario 2: API-based Enrichment

### Input
Tourism dataset + external API data

### Process
- Compute KPIs from CSV
- Fetch weather data from API
- Combine both

### Output
Report including:
- Tourism KPIs
- Weather conditions

### Justification
Weather conditions influence tourism demand:
- High temperature increases beach tourism
- Bad weather reduces travel activity
